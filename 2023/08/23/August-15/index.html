# Maybe

# today

现在是第三天晚上凌晨一点了，前天晚上打算写些的，但最终没写什么，我给你发消息，你为什么总不回啊，明明都可以刷抖音，你说看电视手机在旁边，你看睡着了我都能接受，但你这样我好像感受不到你的用心，我会没收到你的消息会打开微信上下滑动看看有没有提示，和你说了晚安，跟你讲的话你第二天可能看了也没当回事，你不会回答我晚上问的问题，也不会说别的话，发表情包和解释一下可能就是你最大的程度了，但俩个人想要发展是不是应该用点心思，我想发消息给你，每次点进消息框往上翻翻，看着你几个字的回话，真是没劲，你追电视能追到熬夜早上四五点，以前和我聊天到时间就睡了，我傻乎乎的有时候你睡不着觉得和你聊天，会让你不觉得失眠煎熬，后来发现只是我自己觉得罢了，原来你不是不能熬夜，只是我不值得，我好想培养好这段关系，但你的回答永远都这么有礼貌，告诉我，我们是普通朋友，太过亲昵的话现在我也不说了，说出口让我难堪，你也漠视，哪有人能接受没有回应的热情，有的话那也是不喜欢，我要喜欢的人奔我而来，告诉我选择的是我，其实吧，以前我有这样的经历，但因为一些原因，我和她分开了，所以我在意双方更深层次有所了解，再做出决定，这样不让彼此都消耗情绪，现在的你让我一个人困在自己的情绪里啦，所以，我不会和你发晚安了，我的晚安只会跟有回应的人说，等那个会分享给我她所有故事的人，我倾听她的糟糕、难过、高兴和惊喜，陪她走到想要去的地方或许才是我想要。我想或许你是喜欢我的，但时间久了没见面，你就失去新鲜感了吧，我好不喜欢这种，或者我想错了，你不善表达，这是我给你想的理由，不知道是不是真的，但好像都不重要了，因为对你来说告诉我你的心意都要我揣测很久，你就不能告诉我你喜欢我，不说长久，哪怕有那么一刻你觉得我对于你是不一样的。

# About tomorrow ☀

那天在地铁站随手拍了张照片，回去翻相册时发现，“幸福穿梭”, 那一刻感觉好奇妙啊，想告诉你，你知道吗，我在梦里梦到我们以后的生活，那是一片山野边，很辽阔很辽阔的地方，天上云趴在远处的山顶旁，后面是一大片一大片的草，风吹的草像一边倒，有一张长椅，我们靠在一起，我笑的很开心，那时候好像我和你都大了些，镜头后的人让我们看着前面，快门声还没亮起，我就睁开了眼，这种回忆可能你会觉得不可能吧哈哈，但它清晰的留在我的脑海里，后来想再进去梦里，就再也回不去了。毕竟，这次我离幸福这么近，这次真的穿梭到了另一个世界，人们都说有无数平行世界，那假如有，那里的我会让你喜欢吗，

# sea🌊🌊

不知道你还记得吗，放假前我和你说过我想去海边，我说我没看过真正的大海，想看下浩瀚之前是什么感受，后来我真的来了，海的感觉真是让人难以比拟，人们都想在海边拍下照片留作纪念，我也不例外，这片海你也来过吧，你有拍下照片留念吗，海和别的事物相比也更容易引起人们的思念，那种壮阔的感觉比瀑布更让人震撼，那种海浪冲上礁石的场景同样令人着迷，我说要到海边，我来了，比我想象的更让我喜欢，我还说过我喜欢你，要和你在一起，你还记得吗哈哈，以前我说的事可都做到了，但这次可能不行了，以后让我看看什么样的人你会喜欢吧，可别选个太差的，让我觉得你眼光不行，那我要失望了。你要让他和你一起追剧熬最长的夜，一起牵着手到自习室学习，一起骑着电瓶车去打球，回去能给你烧好吃的菜，准备你想吃很久的甜品，给你买每天想喝上的奶茶，去看最新上映的电影，去攀爬想爬越的山峰，到山顶看清晨刚升起的日出，到海边看傍晚的落日燃起整片天空，去热闹的街道上看人山人海，去清冷的古道街边偷偷拥抱，走过你想去的一座又一座城市，去最南边的城市感受热带的气息，去最北边的地方体验寒冬，你要去江南的水城，去大雪覆盖的宫殿古迹，去任何你想去的地方，我希望你的生活充实但不忙碌，不要再天天一个人看剧打发时间，让自己生活变得重复，去更值得你的生活里去吧，找到那个能点缀你生活的人，我也想看看什么样才是标准答案，但我觉得他可能会更优秀，但没人会比我对你更好更喜欢你，
